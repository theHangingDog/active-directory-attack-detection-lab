# Active Directory Attack & Detection Lab

Comprehensive documentation of 50+ AD attack techniques and 38+ detection strategies.

## Overview

This lab covers real-world Active Directory attacks and how to detect them. 98 pages of technical content organized by attack type, with practical examples and detection methods.

## What's Included

- **50+ Attack Techniques** - Step-by-step exploitation methods
- **38+ Detection Strategies** - How to identify and stop attacks
- **160+ Screenshots** - Visual walkthroughs of each technique
- **Splunk/KQL Queries** - Detection rules for SIEM tools
- **Windows Event Log Analysis** - Event IDs and what they mean

## Modules

- **M1: Initial AD Exploitation** - LLMNR poisoning, SMB enumeration
- **M2: AD Post-Enumeration** - BloodHound, PowerView, LDAP enumeration
- **M3: DACL Abuse** - 15 different permission-based attacks
- **M4: Kerberos Exploitation** - AS-REP roasting, kerberoasting, delegation attacks
- **M5: Credential Dumping** - NTDS.dit, DCSync, GMSA, SAM hives
- **M6: Ticket Attacks** - Golden, silver, diamond, and sapphire tickets

## Tools Covered

**Attack Tools:**
Responder, BloodHound, PowerView, Impacket, Hashcat, Mimikatz, Kerberoast

**Detection Tools:**
Windows Event Viewer, Sentinel, KQL, Sysmon, Sigma rules
